## Usage

Clone the Repository:
  ```sh
  git clone https://github.com/Tybenson0/Volkswagoneer.git

  ```
Change directory into the project folder
  ```sh
  cd Volkswagoneer

  ```
Run the project
  ```sh
  mvn spring-boot:run

  ```
Go to http://127.0.0.1:8080
